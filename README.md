# Task Manager (CLI)

This is a simple command-line task manager built with Python.
It allows users to add, view, complete, and delete tasks.
All tasks are saved automatically using a JSON file.

## Features
- Add tasks
- View tasks
- Mark tasks as completed
- Delete tasks
- Task statistics

## How to run
1. Make sure Python is installed
2. Open the project folder
3. Run this command:

python task_manager.py
